The Telos name, logo, and app icon are trademarks of **Haven360 Labs**.

The Telos source code is released under the [MIT License](LICENSE). The MIT License governs **software** only. It does **not** grant permission to use Haven360 Labs trademarks.

## Official Telos

**Haven360 Labs** is the steward of the Telos project. An **official Telos** app (Mac App Store distribution, iCloud sync, and supported updates) is planned but **not yet available**. Until then, the supported way to run Telos is to build from this repository or install a release published by Haven360 Labs on [GitHub Releases](https://github.com/Haven360-Labs/telos/releases).

## Permitted uses

You may:

- State that your project is “based on Telos” or “a fork of Telos”
- Refer to Telos to describe compatibility (e.g. “imports Telos CSV export”)
- Use the Telos name in truthful, descriptive text (not as your product name)

## Uses that require permission

You may **not**, without written permission from Haven360 Labs:

- Name your app or distribution **Telos**, or use a confusingly similar name
- Use the Telos logo, app icon, or marketing assets in a way that suggests endorsement
- Imply that your build is the official Telos app or is affiliated with Haven360 Labs

## Forks and community builds

If you distribute a modified version of this repository:

1. Use a **distinct product name** (not “Telos”)
2. Use a **distinct app icon**
3. Include the MIT license and copyright notice from the upstream project
4. Make clear that your build is **unofficial** and not supported by Haven360 Labs

## Questions

Open a [GitHub issue](https://github.com/Haven360-Labs/telos/issues) for trademark questions.
